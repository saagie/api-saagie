Exporting project environment variables
---------------------------------------

**saagieapi.env_vars.export**

Example :

.. code:: python

   saagieapi.env_vars.export(project_id="50033e21-83c2-4431-a723-d54c2693b964", 
                             output_folder="./output/env_vars/",
                             error_folder="./output/error/",
                             project_only=True)
