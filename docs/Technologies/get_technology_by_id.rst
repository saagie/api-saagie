Getting technology name by technology ID
------------------------

**saagieapi.get_technology_name_by_id**

Example :

.. code:: python

   saagieapi.get_technology_name_by_id(technology_id="11d63963-0a74-4821-b17b-8fcec4882863")

.. code:: python

   ('Saagie', 'Jupyter Notebook')
