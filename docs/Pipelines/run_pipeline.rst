Running a pipeline
------------------

**saagieapi.pipelines.run**

Example :

.. code:: python

   saagieapi.pipelines.run(pipeline_id="ca79c5c8-2e57-4a35-bcfc-5065f0ee901c")

Response payload example :

.. code:: python

   {
       "runPipeline":{
           "id":"975253ea-1b91-4633-acdf-dd9b09d53b18",
           "status":"REQUESTED"
       }
   }
