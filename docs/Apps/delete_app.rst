Deleting an app
---------------

**saagieapi.apps.delete**

Example :

.. code:: python

   saagieapi.apps.delete(app_id="a6de6956-4038-493e-bbd3-f7b3616df39e")

Response payload example :

.. code:: python

   {
       'deleteApp': {
           'id': 'a6de6956-4038-493e-bbd3-f7b3616df39e'
       }
   }
