Running a job
-------

**saagieapi.jobs.run**

Example :

.. code:: python

   saagieapi.jobs.run(job_id="f5fce22d-2152-4a01-8c6a-4c2eb4808b6d")

Response payload example :

.. code:: python

   {
       "runJob":{
           "id":"5b9fc971-1c4e-4e45-a978-5851caef0162",
           "status":"REQUESTED"
       }
   }
