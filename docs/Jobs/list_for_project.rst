Listing jobs for a project
-------------

**saagieapi.jobs.list_for_project**

Example :

.. code:: python

    saagieapi.jobs.list_for_project(project_id="860b8dc8-e634-4c98-b2e7-f9ec32ab4771",
                                    instances_limit=2)
